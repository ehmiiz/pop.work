# pop.work
Specific configs I use for Pop!_OS and my [FrameWork laptop](https://frame.work)

- pwshInstall
    - `Installs PowerShell`
- egpuSwitcherInstall
    - `Installs EGPU-Switcher via git repo and runs setup `
- fingerPrintInstall
    - `Installs the frame.work fingerprint sensor`
- edgeInstall
    - `Install MSEdge`
